# IDTC — Invest Energy-Development Coin

Official whitepaper website for **IDTC Energy Capital** — blockchain-based energy investment platform on Polygon.

**Live site:** https://idtc.online

## Deployment

Auto-deploys to production on push to `main` via GitHub Actions.

**Required Secrets** (shared with xmanstudio repo):
- `SSH_HOST`, `SSH_USER`, `SSH_PRIVATE_KEY`, `SSH_PORT`

**Deploy path:** `/home/admin/domains/idtc.online/public_html`
